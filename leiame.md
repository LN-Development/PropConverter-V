# PropConverter-V

Add-on para conversão automática de props para GTA V.

## Dependências

[Sollumz](https://github.com/Sollumz/Sollumz) 2.8.0

## Instalação

1. Não extraia o arquivo .zip após o download.
2. Abra o Blender.
3. Vá em Edit > Preferences > Add-ons > Install from Disk.
4. Localize o arquivo .zip do add-on e selecione-o.
5. Instale e reinicie o Blender.
